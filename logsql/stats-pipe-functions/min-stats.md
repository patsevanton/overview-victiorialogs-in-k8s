### min stats

Функция конвейера статистики `min(field1, ..., fieldN)` возвращает минимальное значение среди всех указанных `полей логов`.

Например, следующий запрос возвращает минимальное значение поля `duration` за последние 5 минут логов:

```logsql
_time:5m | stats min(duration) min_duration
```

Функция `min(some_field)` работает и со строковыми значениями поля `some_field`, поэтому она возвращает пустую строку, если `some_field` отсутствует в некоторых обрабатываемых логах согласно `модели данных VictoriaLogs`.
Используйте синтаксис
`min(some_field) if (some_field:*) as min_value_without_empty_string`
для фильтрации пустых строковых значений. Подробнее см. в `документации по условной статистике`.

Также можно найти минимальное значение среди всех полей с общим префиксом, используя синтаксис `min(prefix*)`.

Функция ``row_min`` может использоваться для получения других полей, соответствующих минимальному значению `duration`.
