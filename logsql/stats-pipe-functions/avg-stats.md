### avg stats

Функция конвейера статистики `avg(field1, ..., fieldN)` вычисляет среднее значение по всем указанным `полям логов`.
Нечисловые значения игнорируются. Если все значения нечисловые, возвращается `NaN`.

Например, следующий запрос возвращает среднее значение поля `duration` за последние 5 минут:

```logsql
_time:5m | stats avg(duration) avg_duration
```

Также можно вычислять среднее по полям с общим префиксом, используя синтаксис `avg(prefix*)`. Например, следующий запрос вычисляет среднее
по всем полям логов с префиксом `foo`:

```logsql
_time:5m | stats avg(foo*)
```
