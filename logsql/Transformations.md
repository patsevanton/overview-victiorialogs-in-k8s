## Преобразования

LogsQL поддерживает следующие преобразования для записей журнала, отобранных с помощью `фильтров`:

- **Извлечение произвольного текста** из `полей журнала` по заданному шаблону.  
  Подробнее см. в `документации по оператору `extract``.

- **Распаковка полей JSON** из `полей журнала`.  
  См. `документацию по оператору `unpack_json``.

- **Распаковка полей в формате logfmt** из `полей журнала`.  
  См. `документацию по оператору `unpack_logfmt``.

- **Распаковка сообщений Syslog** из `полей журнала`.  
  См. `документацию по оператору `unpack_syslog``.

- **Создание нового поля** на основе существующих `полей журнала` по заданному формату.  
  См. оператор ``format``.

- **Замена подстрок** в указанном `поле журнала`.  
  См. документацию по операторам ``replace`` и ``replace_regexp``.

- **Создание нового поля** на основе математических вычислений над существующими `полями журнала`.  
  См. оператор ``math``.

Также см. `другие операторы (pipes)`, которые можно применять к отобранным записям журнала.

Кроме того, можно выполнять различные преобразования **на стороне клиента** для `отобранных записей журнала` с помощью утилит командной строки Unix: `jq`, `awk`, `cut` и др.  
Подробнее см. в `разделе о запросах из командной строки`.
