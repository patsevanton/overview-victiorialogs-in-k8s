### Удалить поля (delete pipe)

Если необходимо удалить некоторые `поля логов`, можно использовать конвейер `| delete field1, ..., fieldN` `pipe`.

Например, следующий запрос удаляет поля `host` и `app` из логов за последние 5 минут:

```logsql
_time:5m | delete host, app
```

Для удобства вместо `delete` можно использовать ключевые слова `drop`, `del` и `rm`. Например, `_time:5m | drop host` эквивалентно `_time:5m | delete host`.

Можно удалять поля с общим префиксом. Например, следующий запрос удалит все поля с префиксом `foo`:

```logsql
_time:5m | delete foo*
```
