## Фильтр по времени

Сужает область поиска по времени. Рекомендуется использовать для оптимизации запросов.

**Примеры:**

```logsql
_time:1h AND error
_time:5m
_time:2.5d15m42.345s
_time:2023-04-25Z
_time:[2023-04-01Z, 2023-04-30Z]
_time:5m offset 1h
_time:2023-04-25+05:30
_time:>2023-04-01Z
_time:<=2023-04-30Z
```
