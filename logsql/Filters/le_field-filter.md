### Фильтр `le_field`

Иногда требуется найти логи, в которых значение одного `поля` не превышает значение другого поля.  
Это можно сделать с помощью фильтра `field1:le_field(field2)`.

Например, следующий запрос отбирает логи, в которых значение поля `duration` `поля` не превышает значения поля `max_duration`:

```logsql
duration:le_field(max_duration)
```

Полезный совет: используйте `NOT duration:le_field(max_duration)`, чтобы найти логи, в которых `duration` превышает `max_duration`.
