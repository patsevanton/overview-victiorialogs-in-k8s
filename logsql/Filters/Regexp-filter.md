## Фильтр по регулярным выражениям

Поддерживает фильтрацию по регулярным выражениям (синтаксис RE2) через конструкцию `~"regex"`. По умолчанию применяется к полю `_msg`.

**Примеры:**

```logsql
~"err|warn"
~"(?i)(err|warn)"
~'"foo":"(bar|baz)"'
~"a\\.b"
event.original:~"err|warn"
"event:original":~"err|warn"
```
