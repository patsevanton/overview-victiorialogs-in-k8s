### Фильтр по подстроке

Если требуется найти логи, содержащие определённую подстроку, можно использовать фильтр `*подстрока*`. При необходимости подстроку можно заключить в кавычки — см. `документацию`.

Например, следующий запрос отбирает записи логов, в которых в поле ``_msg`` встречается текст `ampl`:

```logsql
*ampl*
```

Этот фильтр подойдёт для таких сообщений:

- `Example message`
- `This is a sample`

Но он **не подойдёт** для `EXAMPLE message`, поскольку подстрока `AMPL` здесь записана заглавными буквами. В таком случае используйте фильтр ``~"(?i)ampl"`` — он учитывает регистр нечувствительно. Обратите внимание, что фильтр без учёта регистра может работать значительно медленнее, чем чувствительный к регистру.

**Совет по производительности:** отдавайте предпочтение `фильтру по слову` и `фильтру по фразе`, поскольку фильтр по подстроке может работать довольно медленно.
