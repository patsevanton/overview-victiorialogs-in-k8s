### Фильтр по слову — кратко

* Запрос из одного слова ищет это слово в поле `_msg`:

  ```
  error
  ```

* Соответствует строкам, содержащим **точное совпадение** слова `error`.

* Не соответствует:

  * `ERROR` — поиск чувствителен к регистру. Без регистра: `i(error)`.
  * `errors` — ищется точное слово. Для префикса: `error*`.

* Чтобы искать в конкретном поле:

  ```
  log.level:error
  ```

* Если имя поля или слово содержит спецсимволы — заключайте в кавычки:

  ```
  "ip:remote":"1.2.3.45"
  ```
