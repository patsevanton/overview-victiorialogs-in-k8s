### Фильтр дневного диапазона

`_time:day_range[start, end]` — выбирает логи в интервале `hh:mm … hh:mm` **каждый день**.

Пример (08:00–18:00 UTC, начало включено, конец исключён):

```logsql
_time:day_range[08:00, 18:00)
```

Границы:

* `[` / `]` — включить время
* `(` / `)` — исключить время

Пример (исключить 08:00, включить 18:00):

```logsql
_time:day_range(08:00, 18:00]
```

Указание часового пояса UTC+2:

```logsql
_time:day_range[08:00, 18:00) offset 2h
```
